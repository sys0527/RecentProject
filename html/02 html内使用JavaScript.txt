在head里加 <script></script>
<script>里面写code
重要** document.write("" or 方程式)

完整的Javascript应该由下列三个部分组成:

核心(ECMAScript)——核心语言功能

eg:

<!script type="text/javascript" src="app.js">

//

<head>
	<!script用来写入javascript>
	
	<script>
		suger = 4;
		num = 3;
		document.write(suger*num)
	</script>
</head>

//

<script>
	function calc(suger,num){
		return suger*num;
	}
	function printResult(suger,num){
		document.write(calc(suger,num));
	}
	printResult(5,5)
</script>

文档对象模型(DOM)——访问和操作网页内容的方法和接口
html访问js 文件（里面有代码可改变html显示的内容）
ex：
app.js
var para = document.getElementById("para");
para.style.color="blue";

html访问css文件（里面的风格是这样）
style.css
.para2{
    font-size : 22px;
    color: #f00;
    text-align : 20px;
    padding-left: 20px;
}
.para{
    font-size : 44px;
    color: #3ed;
    text-align : center;
    padding-left: 2em;
}
